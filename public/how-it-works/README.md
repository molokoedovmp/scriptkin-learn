# Скриншоты для страницы «Как это работает»

Страница автоматически подхватит изображения с этими именами:

- `sql-editor.webp` — редактор с написанным запросом и видимой подсветкой.
- `task-bank.webp` — список заданий «Полуночного экспресса» с фильтрами.
- `quest-progress.webp` — карта шагов квеста или экран прогресса.

Рекомендуемый формат: WebP, ширина 1200–1600 px, соотношение сторон примерно
16:10. Лучше снимать только область продукта без вкладок и панели браузера.
Пока файла нет, вместо него отображается встроенный аккуратный макет.
