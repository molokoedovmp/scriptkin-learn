## 1. Кратко

Ночной экспресс Москва — Петербург. В 03:12 пассажира из купе 7 — торговца антиквариатом Виктора Ланского — находят мёртвым. Врач из соседнего купе говорит: сердечный приступ, умер около половины третьего. Детектив транспортной полиции, который едет этим же поездом в отпуск, проверяет факты по базе данных поезда — и шаг за шагом выясняет, что это убийство.

**Развязка:** убийца — врач Софья Белова, которая всю ночь «помогает» следствию. Её отца когда-то осудили за кражу, которую на самом деле организовал Ланской. Её выдаёт время смерти, которое она назвала сама: по журналу проводника Ланской был жив на одиннадцать минут позже.

## 4. Система улик

1. **Время смерти.** Белова говорит: 02:30. Журнал проводника: в 02:41 Ланской просил чай. Противоречие впервые звучит из уст проводника в сцене осмотра, окончательно вскрывается на уроке 5.
2. **След укола.** Детектив видит его при осмотре; Белова объясняет «капельницей от давления» и просит поверить на слово.
3. **Тамбур, 02:58.** Хлопок двери на полном ходу + осколок ампулы у порога (сцена «Тамбур») + слова Штерн: видела Белову, «у неё что-то блестело в руке» (сцена после урока 7).
4. **Билет, купленный последним** — за день до отправления, в вагон 4 (внимательный игрок увидит сам: `SELECT * FROM tickets ORDER BY purchased_at`).
5. **Фамилия.** Реставратор Андрей **Белов** из старого дела — и Софья **Белова** (отчество «Андреевна» дважды звучит в сценах до финала).
6. **Подсчёт (урок 8).** Все 12 пассажиров на месте — никто не сходил в Бологом. Версия побега, которую продвигала Белова, не подтверждается.

Ложные следы: Штерн (безбилетница с доступом и мотивом), профессор Ветров (связан со старым делом), версия «покупатель отомстил за подделки» и версия «убийца сошёл в Бологом» (обе озвучивает сама убийца), Крамер (коллекционер рядом с антикваром).

## 5. Структура квеста

```
Сцена 0 «Пролог» (5 кадров)
  → Урок 1. Список подозреваемых      (SELECT *)
Сцена 1 «Вагон номер четыре» (4 кадра)
  → Урок 2. Купе номер семь           (WHERE + AND)
Сцена 2 «Осмотр купе» (5 кадров)
  → Урок 3. Билет в один конец        (WHERE по внешнему ключу)
Сцена 3 «Сосед по вагону» (4 кадра)
  → Урок 4. Маршрут в ночи            (ORDER BY)
Сцена 4 «Штабное купе» (4 кадра)
  → Урок 5. Журнал проводника         (BETWEEN)
Сцена 5 «Тамбур» (4 кадра)
  → Урок 6. Кофр с двойным дном       (numeric, выбор колонок)
Сцена 6 «Опись багажа» (5 кадров)
  → Урок 7. Лишний пассажир           (LEFT JOIN + IS NULL)
Сцена 7 «Пассажирка без билета» (5 кадров)
  → Урок 8. Пересчитать всех          (GROUP BY + COUNT)
Сцена 8 «Разговор с Беловой» (5 кадров)
  → Финал «Дело закрыто»
```

Тексты уроков (главы, задания, «развития») — см. `db/quests/midnight-express.steps.sql`; они дублируют сцены только там, где игрок мог сцену пропустить.

## 7. Завязка и уроки

Тексты завязки («Материалы дела»), восьми уроков (глава → задание → подсказка → развитие) и финала хранятся в `db/quests/midnight-express.steps.sql` — они согласованы со сценами: сцены дают живой диалог, уроки — работу с данными и выводы.

Краткая канва уроков:

| Урок | SQL-тема | Что даёт сюжету |
|------|----------|-----------------|
| 1. Список подозреваемых | `SELECT *` | 12 пассажиров; в вагоне 4 — врач, коллекционер, переводчица и жертва |
| 2. Купе номер семь | `WHERE + AND` | Личность жертвы; старое дело о краже; эксперт Ветров в этом же поезде |
| 3. Билет в один конец | `WHERE` по FK | Ланской ехал до конца; чей-то билет куплен за день до отправления |
| 4. Маршрут в ночи | `ORDER BY` | Ночные стоянки; Белова продвигает версию побега через Бологое |
| 5. Журнал проводника | `BETWEEN` | Чай в 02:41 против «смерти в 02:30»; дверь тамбура в 02:58 |
| 6. Кофр с двойным дном | numeric | Подделки вместо коллекции; запах масла объяснён; женщина без посадки |
| 7. Лишний пассажир | `LEFT JOIN + IS NULL` | Штерн — безбилетница; её показание против Беловой |
| 8. Пересчитать всех | `GROUP BY + COUNT` | Никто не сошёл — версия побега уничтожена; улики сходятся на Беловой |

## 8. Как заменить картинки на свои

1. Положи файлы в `public/quests/midnight-express/` (например `wagon4.png`, `kramer.png`, `hq.png`, `luggage.png`, `finale.png`).
2. В `db/quests/midnight-express.scenes.sql` пропиши `image_url` нужных кадров: `/quests/midnight-express/wagon4.png`.
3. Примени файл к базе: `psql $DATABASE_URL -f db/quests/midnight-express.scenes.sql`.
4. Обнови страницу квеста — сцены читаются из базы при каждом открытии.
